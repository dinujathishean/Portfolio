================================================================================
VERCEL — STATIC SITE AT REPO ROOT (works out of the box)
================================================================================

This repo has **index.html** at the **repository root** (same level as README.md).
No subfolder, no rewrites required.

STEPS
-----
1. https://vercel.com → Sign in with GitHub.

2. **Add New… → Project** → Import **dinujathishean / Portfolio**.

3. **Configure:**
   - **Framework Preset:** Other
   - **Root Directory:** **.**  (leave default — do NOT type "project")
   - **Build Command:** empty
   - **Output Directory:** empty

4. **Deploy.**

5. Open the **.vercel.app** URL — you should see the portfolio home page.

IF YOU ALREADY CREATED THE PROJECT WITH WRONG SETTINGS
-------------------------------------------------------
**Project → Settings → General → Root Directory** → set to **.** → Save.

**Deployments →** latest → **⋯** → **Redeploy** (disable "Use existing Build Cache"
if the old deploy was broken).

IF THE SITE IS STILL BLANK
---------------------------
- Push the latest **main** branch to GitHub (must contain **index.html** at root).

- In Vercel: **Deployments** → open the build → read the log for errors.

- Try an incognito window or another browser (cache).
